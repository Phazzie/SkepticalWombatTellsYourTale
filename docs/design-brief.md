# Design Brief

## Tone

Serious, warm, and focused. This is a voice-first writing studio, not a social feed or playful dashboard.

## Visual Personality

- Clean, modern dark interface
- High information clarity and hierarchy
- Subtle depth and glow accents
- Minimal visual noise

## Pizzazz Constraints

- Micro-interactions only
- No novelty effects
- No distracting motion
- Motion must support feedback and focus

## Accessibility Baseline

- Keyboard operable flows
- Visible focus rings
- Reduced motion support
- High-contrast text on dark surfaces

## Performance Budget

- Keep runtime lightweight
- No heavy animation libraries
- Prefer simple CSS transitions and reusable primitives
