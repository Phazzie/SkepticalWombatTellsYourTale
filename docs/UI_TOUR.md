# UI Tour

This document describes the primary user flows and provides a checklist for UI screenshots.

## Primary flows

1. **Sign in / Register**
   - `/sign-in`
   - `/register`

2. **Project list & create**
   - `/` (create a new project, open an existing one)

3. **Record a session**
   - `/project/[id]/record`
   - Flow: start recording → transcription → AI analysis panels

4. **Review sessions**
   - `/project/[id]/sessions`
   - Expand a session to view transcript + AI annotations

5. **Documents & voice draft**
   - `/project/[id]/documents`
   - Flow: create document → edit content → “Generate in your voice” draft

6. **Questions**
   - `/project/[id]/questions`
   - Filter questions and answer via “Answer →” recording flow

7. **Export**
   - `/project/[id]/export`
   - Select export level and download Markdown

## Dashboard widgets (project overview)

- Dropped threads (tangents)
- Gaps
- Patterns
- Questions summary
- Concept candidates
- Open contradictions
- Search across everything
